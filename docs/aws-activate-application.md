# AWS Activate Application Notes for Eligibil.org

## Startup description
Eligibil.org is an AI-powered funding intelligence and eligibility matching platform that helps startups, SMEs, NGOs, researchers and founders discover relevant grants, accelerators, EU programs, non-dilutive capital, investment opportunities, training programs and ecosystem events.

## Product
The platform aggregates funding and ecosystem opportunities, structures eligibility criteria, and uses AI-assisted matching to recommend relevant opportunities based on startup profile, sector, location, stage, funding need and readiness.

## Target users
- Early-stage startups
- SMEs
- NGOs
- Researchers
- Founders
- Incubators and ecosystem builders

## AWS credits use case
AWS credits will be used to build and scale the MVP infrastructure, including hosting, secure storage, database, automated data ingestion, search, AI-assisted eligibility scoring, document processing, email notifications, monitoring and analytics.

## Planned AWS services
- Amazon S3
- Amazon CloudFront
- AWS Lambda
- Amazon API Gateway
- Amazon RDS PostgreSQL or DynamoDB
- Amazon OpenSearch
- Amazon Bedrock
- Amazon SES
- Amazon EventBridge
- Amazon CloudWatch
- AWS Budgets

## Application answer draft
Eligibil.org will use AWS credits to build and scale the MVP infrastructure for an AI-powered funding intelligence and eligibility matching platform. The credits will support hosting, secure user profiles, funding opportunity ingestion, search, AI-assisted eligibility scoring, document processing, recommendations, notifications, analytics and monitoring. We plan to use AWS services such as S3, CloudFront, Lambda, API Gateway, RDS or DynamoDB, OpenSearch, SES, EventBridge, CloudWatch and Amazon Bedrock.

## Compliance note
Eligibil.org does not guarantee funding approval. The platform helps users discover, compare and evaluate opportunities. Users must verify final eligibility and application requirements with the official funding provider.
