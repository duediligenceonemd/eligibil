## One-line description

## Short startup description
Eligibil.org helps startups, SMEs, NGOs, researchers and founders discover relevant grants, accelerators, EU programs, non-dilutive capital, investor opportunities, training programs and ecosystem events. The platform structures funding opportunities and eligibility criteria, then uses AI-assisted matching to recommend relevant opportunities based on profile, sector, location, stage, funding need and readiness.

## Problem
Founders and small organizations lose significant time searching across fragmented grant portals, accelerator websites, EU calls, public programs and ecosystem resources. Eligibility criteria are hard to compare, deadlines change, and teams often do not know which opportunities fit their profile before investing time in an application.

## Solution
Eligibil.org centralizes opportunity discovery, structures eligibility data and helps users compare relevant programs through AI-assisted matching. The roadmap includes eligibility scoring, document and pitch deck readiness support, deadline alerts, recommendations and automated opportunity ingestion.

## Current traction / status
Eligibil.org is currently an MVP / early access product in development. The public website, funding search, resource catalog, health checks, analytics and foundational legal/technology pages are live. Advanced AI scoring, document readiness and recommendation workflows are in roadmap development.

## AWS credits use case
AWS credits will be used to build and scale the MVP infrastructure for an AI-powered funding intelligence and eligibility matching platform. The credits will support hosting, secure user profiles, funding opportunity ingestion, search, AI-assisted eligibility scoring, document processing, recommendations, notifications, analytics and monitoring.

## Planned AWS services
- Amazon S3 for document and asset storage
- Amazon CloudFront for fast content delivery
- AWS Lambda for serverless workflows
- Amazon API Gateway for backend APIs
- Amazon RDS PostgreSQL or DynamoDB for structured data
- Amazon OpenSearch for semantic and faceted opportunity search
- Amazon Bedrock for AI-assisted matching, summarization and eligibility scoring
- Amazon SES for email notifications
- Amazon EventBridge for scheduled data ingestion
- Amazon CloudWatch for monitoring
- AWS Budgets for cost control

## Why AWS
AWS is a strong fit because Eligibil.org needs scalable storage, scheduled data ingestion, secure APIs, managed databases, search infrastructure, AI capabilities, email delivery, monitoring and cost controls. AWS credits would allow the project to build the MVP infrastructure responsibly while validating demand and product usage.

## Compliance / disclaimer
Eligibil.org does not guarantee funding approval. The platform helps users discover, compare and evaluate opportunities. Users must verify final eligibility and application requirements with the official funding provider.
