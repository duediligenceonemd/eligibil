Technology Stack & Infrastructure Roadmap

Eligibil.org is an AI-powered funding intelligence platform for startup founders. Below is the current technology stack powering the MVP and the planned cloud infrastructure for scaling.

Current stack (MVP — live)

LayerTechnology
ServerNode.js + Express 4
DatabaseSupabase PostgreSQL + pgvector for semantic search
FrontendVanilla HTML/JS, Preact components (CDN)
Authbcrypt + express-session (custom, server-side)
AI — EmbeddingsAWS Bedrock Titan Embed V2 / OpenAI text-embedding-3-small (grant vectorization)
AI — ExtractionAWS Bedrock Claude / Anthropic Claude API (grant parsing, enrichment)
EmailResend (transactional, onboarding, deadline alerts)
Searchpgvector cosine similarity + full-text fallback
HostingGoogle Cloud Run (containerized)
i18n4 languages: Romanian, English, Russian, Ukrainian
MonitoringSentry (errors + profiling), Google Analytics, Microsoft Clarity

AI use cases (current + planned)

  • Document analysis — parsing and scoring pitch decks (PDF/PPTX), pitch videos (transcription + scoring), whitepapers and websites against funding eligibility criteria
  • Eligibility matching — semantic comparison between founder profiles and the structured database of 700+ funding programs
  • Recommendation generation — natural language gap analysis and improvement recommendations per funding opportunity
  • Grant extraction — automated parsing of funding program announcements from emails, web pages and documents
  • Multi-language support — operating in Romanian, English, Russian and Ukrainian for the target markets

Planned AWS infrastructure (roadmap)
