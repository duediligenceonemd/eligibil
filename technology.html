Technology Roadmap

Eligibil.org is designed as a cloud-native funding intelligence and eligibility matching platform. The current MVP uses a web application, database-backed opportunity catalog, user flows and monitoring. Future infrastructure is planned to support secure profiles, document processing, search, recommendations and AI-assisted scoring.

Planned cloud infrastructure use cases

  • Web application hosting and fast content delivery
  • Secure user profiles and opportunity database
  • AI-assisted matching, summarization and eligibility scoring
  • Document and pitch deck readiness support
  • Search, recommendations, alerts and analytics
  • Scheduled ingestion and monitoring

Planned AWS services

Amazon S3 for document and asset storage
Amazon CloudFront for fast content delivery
AWS Lambda for serverless workflows
Amazon API Gateway for backend APIs
Amazon RDS PostgreSQL or DynamoDB for structured data
Amazon OpenSearch for semantic/faceted opportunity search
Amazon Bedrock for AI-assisted matching and summaries
Amazon SES for email notifications
Amazon EventBridge for scheduled ingestion
Amazon CloudWatch and AWS Budgets for monitoring and cost control

Note: These are planned or roadmap services unless explicitly documented as implemented. Eligibil.org does not claim AWS Activate approval or production AWS usage.
